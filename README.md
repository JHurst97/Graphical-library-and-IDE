# Graphical-library-and-IDE
This project allows users to write code to create 'sketches' similar to that of Processing.


=-=-=-=-=--=EXAMPLE COMMANDS=-=-=-=-=
<p>
	<u>Methods (with parameters):</u>
	method(name,first,second,for,fiv){
	rect(first,second);
	rect(for,fiv);
	};
	name(123,234,345,456);


<u>Methods:</u>
	method(mName, p,a,r,a,m,s){
		print(hello);
	};
	mName();


<u>Variables:</u>
	int(name, 123);

<u>loop: </u>
	loop(5){
	//contents
	};
	
<u>if statement:</u>
	if(1 < 3){
	//contents
	};

<u>Print line:</u>
	print(hello);

<u>Rectangle:</u>
	rect(20,20);

<u>Line:</u>
	line(20,20);

<u>Move pen:</u>
	Position pen(20,20);

<u>Circle:</u>
	ellipse(100,100);

</p>
